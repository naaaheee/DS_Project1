# 📌 DS Project 1 - Subtitle Management System

## 📁 프로젝트 개요

- **과목명**: 데이터구조 (2024년 2학기)
- **주제**: 자막 정보를 다양한 자료구조로 관리
- **언어**: C++
- **구현 환경**: Visual Studio Code (Windows)  
  → 실행 및 테스트는 VMware Ubuntu 환경에서 G++ 컴파일러 사용

## 🔧 주요 기능

- 연결 리스트 기반 섹션 리스트 및 자막 리스트 구현
- 큐(Queue)를 사용한 자막 처리
- 이진 탐색 트리(BST)를 사용해 자막 정렬 및 탐색
- Manager 클래스 기반 명령어 처리 시스템

## 🧠 배운 점

- 큐, 트리, 리스트 구조의 차이점과 구현 방법 숙지
- 클래스 간 협업 설계 및 헤더 분리 구조 학습
- 포인터와 메모리 구조에 대한 이해 향상

## ⚙️ 실행 방법

```bash
cd src
g++ -std=c++11 *.cpp -o subtitle
./subtitle
```
